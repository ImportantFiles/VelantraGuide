3) Onboarding Process

STEP 1: REGISTERING FOR GNT ACCOUNT

  • Do you already have an existing GNT profile account?
  • IF YES:
    • Ask if it was set up through the link or previously existing
    • If existing, submit helpdesk ticket to connect IB link
  • IF NO:
    • Send IB Link
    • Guide client through registration

STEP 2: SUBMIT FOR KYC/KYB VERIFICATION

  • Confirm required documents
  • Ensure exact match across all documents
  • Complete verification steps

STEP 3: SET UP GNT X FOLLOWER ACCOUNT

  • Send offer link
  • Select GNT X Follower
  • Confirm leverage and currency

STEP 4: FUNDING THEIR ACCOUNT

  • Crypto → follow deposit steps
  • Wire → follow instructions + helpdesk if needed

STEP 5–8

  • Wait for agreements
  • Send funds
  • Confirm deposit
  • Activate account

STEP 9

  • Collect account login
  • Collect investor password

4) Post-Call Checklist

  • Add notes in GHL
  • Send recap email
  • Update pipeline
  • Update KeyMaker
  • Set next task or appointment

5) Day-to-Day Checklist

  • Check conversations
  • Send FXBlue link
  • Update KeyMaker
  • Verify account activity
